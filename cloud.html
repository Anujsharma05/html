ABOUT CLOUD
Cloud Computing :- It is a new form of Internet-based computing that provides shared computer processing resources and data to computers and other devices on demand. Types of Cloud :- Software As A Service(SAAS) :- It allows users to subscribe and use application software in the cloud. Storage As A Service(STAAS) :- It is a cloud computing model in which subscribers can rent storage from cloud providers. Infrastructure As A Service(IAAS) :- It is a form of cloud computing that provides virtualized computing resources over the internet. Platform As A Service(PAAS) :- It is a complete development and deployment environment in the cloud, with resources that enable you to deliver everything from simple cloud-based apps to sophisticated, cloud-enabled enterprise applications.
